ESTRUTURA OBRIGATÓRIA DO ARQUIVO DE TEXTO (.txt)
Cada arquivo deve seguir rigorosamente esta ordem de linhas:

Linha 1: Título do Texto
Linha 2: Categoria (Use apenas: clínica, teoria ou ensaio)
Linha 3: Data de Ordenação (Formato: AAAA-MM-DD - Essencial para a ordem)
Linha 4: Data de Exibição (Como aparecerá no site, ex: "Abril 2026" ou "12/04")
Linha 5: Resumo (Um parágrafo curto que aparece antes do botão "ler")
Linha 6 em diante: O corpo do texto completo. 

DICA DE FORMATAÇÃO:
- Para criar parágrafos no corpo do texto, pule uma linha (deixe uma linha em branco).
- Evite aspas curvas (“ ”) copiadas do Word; prefira aspas retas (" ").

---
COMO ADICIONAR NOVOS TEXTOS NO SITE:

1. Abra o arquivo lista.json que está localizado no mesmo diretorio dos textos e index
2. Insira o nome do texto conforme a sequencia que há no texto, adicione uma virgula e preserve as chaves e os colchetes
3. arquivo atual:
{ 
"arquivos" : ["texto_1a.txt", "texto_2.txt", "texto_3.txt"] 
}
4. inserindo o arquivo "texto_4.txt"
{ 
"arquivos" : ["texto_1a.txt", "texto_2.txt", "texto_3.txt", "texto_4.txt"] 
}
